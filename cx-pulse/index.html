How CX Pulse Works
Understanding the analysis methodology
⚡
Criticality Score (0–100)
Based on priority level, keywords in subject/description, status age, and escalation signals. Critical = 75+, High = 50–74, Medium = 25–49, Low = 0–24.
💬
Sentiment Analysis
Scans subject and description for negative keywords (frustrated, urgent, broken, escalate) and positive signals (thank, resolved, great) to classify sentiment.
🗃
Supabase Storage
Results saved to Supabase PostgreSQL with raw JSON preserved in JSONB column. Enables historical queries and trend analysis over time.
📊
Input Formats
Accepts single ticket object or array of tickets. Compatible with Zoho Desk JSON export format. Required fields: id or ticketNumber, subject. All others optional.
